About this project

This is a little enchantement chart/table i made to learn WebDev.
It aim at helping user choose accesories to enchant in the long run (more than 1 success item, to regulate the RNG) for profit.
It is usisng the BDO Market API to get item prices in the 90 last days, on the EU server

Future Features

  • Cron Usage
  • Custom timespan for market prices
  • Prices doing all the enchantement (starting from bases bases)
  • DataCenter selector (EU only currently)
  • Color coding
